# Crelolabs Marketplace

A premium Next.js 15 digital product marketplace for Crelolabs, built with React, TypeScript, Tailwind CSS, Framer Motion, ShadCN-style UI primitives, and Lucide icons.

## Pages

- Homepage
- Store with search, category filtering, product cards, ratings, tags, best-seller and trending metadata
- Product detail pages
- User dashboard
- Checkout experience
- Admin panel
- Blog index and article pages
- About
- Contact
- Privacy Policy
- Terms & Conditions

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
